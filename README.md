# [dotfiles](https://github.com/gortspace/dotfiles)

```
sudo apt install stow

git clone git@github.com:gortspace/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow bash
stow git # Important! not ".git"
stow tmux
stow zsh
stow nvim
```
